# hanspe_check

## Description
MSスタイル日本語半角スペースチェック

## Screen image
![image](https://user-images.githubusercontent.com/10069642/83829999-21232e80-a71f-11ea-8e7f-7519f7cf4a40.png)  

## USAGE
1. hanspe_check.exeを起動する。
2. 半スペチェックしたい*.txt（utf8）をドラッグ&ドロップする。
3. 処理完了後はブラウザに結果が表示される。
